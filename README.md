# Sri Lanka Tours - Tourism Website

A modern tourism website showcasing travel destinations and tour packages in Sri Lanka. Built with React.js and Tailwind CSS.

## Features

- ✅ Fully Responsive Design (Mobile, Tablet, Desktop)
- ✅ Modern UI with Smooth Animations (Framer Motion)
- ✅ Dark Mode Support
- ✅ SEO Optimized
- ✅ Clean Component Structure
- ✅ Reusable Components

## Pages

1. **Homepage** - Hero section, featured destinations, tour packages, testimonials, CTA
2. **Destinations** - Grid of popular travel destinations with search functionality
3. **Tour Packages** - List of packages with filtering and sorting options
4. **Package Details** - Detailed view with image gallery, itinerary, and booking
5. **Contact** - Contact form with Google Maps integration

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Framer Motion (Animations)
- React Icons
- Google Maps Embed API
